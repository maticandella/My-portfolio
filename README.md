## 📌 Sobre este proyecto
Portfolio profesional creado con **Astro + Tailwind CSS**, diseñado para:
- Mostrar proyectos full-stack
- Resaltar experiencia en **.NET Core, Node.js y Angular**
- Componentes reutilizables (Tarjetas de proyectos, Navbar)
